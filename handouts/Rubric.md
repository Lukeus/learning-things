# Assessment Rubric (0–2 each, pass ≥ 9/12)
1. **Clarity of Spec** – behavior, constraints, edge cases.
2. **Use of Clarification** – closed material gaps before planning.
3. **Plan Quality** – boundaries, risks, tests.
4. **Task Granularity** – small, testable, delegable.
5. **Implementation Fidelity** – code matches spec; tests prove it.
6. **Versioned Artifacts** – spec/plan/tasks committed.
